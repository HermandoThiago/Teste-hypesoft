### Tecnologias utilizadas no projeto:

- React
- Typescript
- Styled-components
- React Icons
- Docker

### Como inicializar o projeto?

### Docker-Compose

```
docker-compose up -d
```

### yarn

Instalar as dependências
```node
yarn install
```

Iniciar o ambiente de desenvolvimento
```node
yarn dev
```

Agora só abrir http://localhost:3000/ no navegador
